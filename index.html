How to flash it

  1. Paste the macros { } block just inside the top-level / { … } in config/sofle.keymap (alongside your existing nodes).
  2. Add the &poe_* references onto free &trans keys — your CONFIG layer has plenty of open slots.
  3. Commit & push → GitHub Actions builds two .uf2 files (left + right).
  4. Double-tap reset on a half → it mounts as NICENANO → drag the matching .uf2 on. Repeat for the other half.

If PoE drops characters mid-macro, bump tap/wait ms to 12–15 and regenerate.
